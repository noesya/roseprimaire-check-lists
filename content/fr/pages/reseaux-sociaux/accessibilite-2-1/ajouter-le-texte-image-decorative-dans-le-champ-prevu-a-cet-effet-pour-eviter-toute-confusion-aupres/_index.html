La plupart des réseaux sociaux offrent la possibilité de renseigner des alternatives textuelles aux images directement dans leur interface de rédaction. Cependant, il est important de noter qu’ils ne permettent pas d’identifier une image comme étant décorative.

Sur X (ex-Twitter) : Par défaut, les images postées sur Twitter ont pour alternative « image »

Sur LinkedIn : Dans le cas où ce champ resterait vide, la mention « Aucune description alternative pour cette image » est ajoutée automatiquement.

Compte-tenu des précisions ci-dessus, il est recommandé d’indiquer dans le champ description « image décorative » lorsque c’est nécessaire, de manière à prévenir l’utilisateur que l’image n’est pas porteuse d’information.
