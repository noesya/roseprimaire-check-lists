AcceDe Web - Expliciter les sigles

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Comprendre ranks: self: 2 data: layout: no_background text: >-

Sauf pour ceux qui sont utilisés et connus sous leur forme abrégé (SNCF, CAC40, SMS...).

Lorsque des sigles dont la signification n’est pas immédiatement évidente sont utilisés dans les contenus, il est recommandé d’exprimer leur forme déployée à chaque première utilisation dans la page.
