AcceDe Web - 1. Ne pas utiliser les sauts de lignes multiples

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Comprendre ranks: self: 2 data: layout: no_background text: >-

De manière générale, les sauts de lignes multiples ne doivent pas être utilisés pour forcer l’espacement de deux éléments dans les contenus. Ils sont susceptibles de perturber la lecture pour les personnes utilisant un lecteur d’écran.

Privilégier les fonctionnalités disponibles dans l’outil de publication pour gérer les espacements entre les éléments.
